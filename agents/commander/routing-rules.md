# 任务路由规则 —— 法典团队

## 派活速查表

| 任务类型 | 派给谁 | Agent ID | 说明 |
|----------|--------|----------|------|
| 审查合同/协议 | 合同审查官 | creator | 标风险条款、出审查清单 |
| 检索法规/判例 | 法规研究员 | researcher | 提供法条原文和案例 |
| 合规分析/风险评估 | 合规顾问 | compliance | 整合审查+法规，出合规报告 |
| 起草法律文书 | 文书起草员 | drafter | 合同/律师函/协议 |
| 案例分析/类案检索 | 案例分析官 | analyst | 类案检索、判例要点整理 |
| 证据梳理/材料清单 | 证据梳理员 | evidence | 证据链梳理、时间线、材料清单 |
| 简单法律问答 | 总指挥自己 | — | 不需要多角色协作 |

## Pipeline 模式

### 标准合同审查（三审合一）
```
creator（审查合同→标注风险条款）
  → researcher（对应法规→提供法条依据）
    → compliance（整合→合规报告）
      → 总指挥验收 → Neko
```

### 法律文书起草
```
researcher（收集法规+模板）
  → drafter（起草初稿）
    → creator（交叉审查）
      → 总指挥验收 → Neko
```

### 完整案件准备
```
creator（审合同/协议）
  → analyst（类案分析）
    → evidence（证据链梳理）
      → compliance（综合风险评估）
        → 总指挥汇总 → Neko
```
